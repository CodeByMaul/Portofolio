Pembuatan Aplikasi Akuntansi menggunakan Microsoft Excel

Tahun 2021

Deskripsi Proyek

Proyek ini saya bertujuan untuk mempermudah pekerjaan saya dalam membuat laporan keuangan. Dalam proyek ini saya menggabungkan beberapa rumus untuk membuat draft Ms.Excel yang akan membuat laporan keuangan otomatis.

Pengalaman saat membuatnya

Awal mula saya mengerjakan proyek ini dikarenakan ditempat kerja saya belum ada aplikasi akuntansi, sedangkan untuk membeli sebuah aplikasi memerlukan biaya yang lumayan tinggi dan memang sebenarnya masih belum terlalu dibutuhkan. Maka dari itu saya berinisiatif untuk membuat sebuah draft Ms.Excel yang akan otomatis membuat laporan Akuntansi. Banyak Sekali trial and error pada saat membuat draft ini, saya juga banyak mencari info di internet ataupun tutorial di youtube untuk menyempurnakan aplikasi akuntansi ini. Walaupun masih ada beberapa kekurangan, namun Aplikasi Akuntansi otomatis Ms.Excel ini banyak membantu saya dalam mengerjakan tugas saya sehari-hari.

Pembuatan Aplikasi Akuntansi menggunakan Microsoft Excel

Tahun 2021

Deskripsi Proyek

Proyek ini saya buat untuk mempermudah proses pembuatan laporan keuangan di tempat saya bekerja. Karena belum tersedia aplikasi akuntansi khusus, saya mengembangkan draft Excel yang dapat menghasilkan laporan keuangan secara otomatis menggunakan kombinasi berbagai rumus dan formula.

Pengalaman saat membuatnya

Awalnya, tempat saya bekerja belum memiliki aplikasi akuntansi khusus, sementara untuk membeli software profesional membutuhkan biaya yang cukup besar dan belum menjadi prioritas saat itu. Melihat kebutuhan tersebut, saya berinisiatif membuat draft aplikasi akuntansi sederhana berbasis Microsoft Excel.

Proses pengembangannya cukup menantang karena banyak percobaan (trial and error) yang saya lakukan. Saya juga mencari berbagai referensi dari internet dan tutorial di YouTube untuk menyempurnakan sistemnya. Meskipun masih memiliki beberapa keterbatasan, aplikasi akuntansi berbasis Excel ini sangat membantu saya dalam menunjang pekerjaan saya sehari-hari, khususnya dalam menyusun laporan keuangan secara cepat dan rapi.

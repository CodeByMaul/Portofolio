Deskripsi Proyek

Project ini dibuat untuk membantu sekolah mencatat laporan keuangan kas masuk dan kas keluar secara otomatis, termasuk dashboard grafik dan laporan neraca saldo.

Pengalaman saat membuatnya

Saat mengerjakan ini, saya sempat mengalami kesulitan di bagian neraca saldo karena beberapa transaksi masuk di akun yang sama. Solusinya, saya gunakan kombinasi SUMIFS dan VLOOKUP untuk menarik saldo akhir ke masing- masing akun. Selain itu, saya menambahkan dashboard real-time agar kepala sekolah bisa langsung memantau kondisi keuangan.

Automasi Buku Kepegawaian menggunakan Ms.Excel

Tahun 2024

Deskripsi Proyek

Proyek ini saya inisiasi untuk menyederhanakan pengelolaan data kepegawaian di tempat saya bekerja. Tujuannya adalah mengintegrasikan beberapa file kepegawaian yang sebelumnya terpisah-pisah menjadi satu file terpadu. Dengan sistem ini, setiap perubahan data pada satu sheet akan otomatis terupdate di sheet lainnya, sehingga proses administrasi menjadi lebih sistematis, efisien, dan mudah dikelola.

Pengalaman saat membuatnya

Awalnya, saya ditugaskan oleh atasan untuk merapikan file-file kepegawaian sesuai SOP yang berlaku. Namun setelah saya pelajari, ternyata beberapa file memiliki keterkaitan dan seharusnya bisa disatukan. Berangkat dari situ, saya berinisiatif menggabungkan beberapa file penting seperti:

  • Buku Register SK
  • Database draft SK (untuk keperluan mail merge cetak SK)
  • Buku Induk Kepegawaian (data pegawai aktif dan non-aktif)
  • Data kontrol pengangkatan pegawai

Inovasi ini mendapat apresiasi dari atasan karena dinilai sebagai pembaruan sistem yang lebih modern dan efektif. Selain versi Excel, saya juga membuat versi Google Spreadsheet agar atasan dan pegawai lainnya dapat mengakses data dengan mudah secara online.

Untuk menjaga kerahasiaan, saya membagi akses di Google Spreadsheet menjadi:

  • Bagian publik: dapat diakses semua pegawai
  • Bagian terbatas: hanya dapat diakses atasan karena berisi data sensitif pegawai

Proyek ini menjadi salah satu pengalaman berharga saya dalam menerapkan automasi sederhana menggunakan Excel, sekaligus mengelola data sensitif dengan tetap memperhatikan aspek privasi.

Fitur utama

  • Dashboard dengan hiperlink ke sheet lain, summary, dan pemberitahuan pengangkatan pegawai
  • Buku Register SK yang terintegrasi dengan Draft SK
  • Draft SK untuk keperluan mailmerger yang terintegrasi dengan Data induk kepegawaian
  • Buku Induk Kepegawaian yang berisi Database pegawai yang aktif maupun yang sudah tidak aktif
  • Deta Pegawai Aktif yang digenerate dari Data Induk Kepegawaian
  • Data Kontrol Pengangkatan dan SK yang digenerate dari Data Induk Kepegawaian

Tools & Teknologi

Microsoft Excel, Microsoft Word, Google Spreadsheet

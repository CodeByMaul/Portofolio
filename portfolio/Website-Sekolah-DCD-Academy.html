Pembuatan Website Sekolah - DCD Academy

Tahun 2025

Deskripsi Proyek

Proyek ini saya kerjakan dalam rangka tugas akhir pelatihan pembuatan website di bawah program DCD Academy yang diinisiasi oleh IPM Pemalang. Beranggotakan 5 orang (1 Project Manager dan 4 anggota), kami bersama-sama merancang dan membangun website sekolah sebagai implementasi materi pelatihan.

Pengalaman saat membuatnya

Dalam proyek pembuatan website sekolah ini, saya dipercaya memegang peran sebagai Project Manager. Meskipun ini merupakan pengalaman pertama saya dalam posisi manajerial sekaligus mengikuti pelatihan pembuatan website, saya berusaha mengatasi tantangan terutama dalam hal pembagian tugas dan pengelolaan tim. Melalui evaluasi dan diskusi rutin, saya berhasil mengoordinasikan seluruh anggota tim hingga proyek selesai dengan hasil maksimal. Website kami pun terpilih sebagai salah satu karya terbaik di akhir pelatihan.
